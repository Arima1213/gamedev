## Developer Notes

### Adding New Premium Items

**Process:**
1. Identify need (community request, design gap)
2. Determine category (cosmetic, convenience, expansion)
3. **CHECK: Does this violate P2W principles?**
4. Set price based on value and category
5. Determine tradeable status
6. Check if IRF alternative exists
7. Create item in Roblox Studio
8. Add to premium shop UI
9. Implement purchase handling
10. Track analytics (conversion, revenue)

**Pricing Guidelines:**

| Item Type | Price Range | Examples |
|-----------|-------------|----------|
| **Small Cosmetics** | 100-300 Robux | Hats, accessories, small decorations |
| **Major Cosmetics** | 300-600 Robux | Full outfits, animated decorations, mounts |
| **Tool Skins** | 200-450 Robux | Weapon/tool visual overrides |
| **Convenience Items** | 300-500 Robux | Automation, crafting stations |
| **Game Passes** | 199-399 Robux | Permanent account upgrades |
| **Value Bundles** | 800-1,500 Robux | 30-50% discount on bundled items |

### P2W Evaluation Checklist

**Before adding ANY Robux item, ask:**

- ‚ùì Can free players obtain similar/equivalent item?
- ‚ùì Does this provide stat advantages?
- ‚ùì Does this lock content behind paywall?
- ‚ùì Is this tradeable (so F2P can obtain via trading)?
- ‚ùì Is this purely cosmetic OR convenience?
- ‚ùì Can free players complete all content without this?

**If ANY answer violates principles ‚Üí DO NOT ADD**
