## Title System Overview

### What Are Titles?

**Definition:**
- Cosmetic text displayed above player name
- NO gameplay effects whatsoever
- Purely for prestige and personalization
- Can be changed at any time

**Display Format:**
```
[Title]
PlayerName
```

Example:
```
[Dragon Slayer]
Steve123
```

### Title Properties

**Key Characteristics:**
- ✅ **Cosmetic Only** - Zero stat bonuses
- ✅ **Collectable** - Unlock and display any owned title
- ✅ **Prestigious** - Some titles very rare
- ✅ **Customizable** - Equip/unequip freely
- ❌ **NOT Tradeable** - Cannot trade titles
- ❌ **NOT Sellable** - Cannot sell titles
- ❌ **NOT Stat Boosts** - No gameplay advantage

### Title Rarity Tiers

| Rarity | How Common | Obtainment | Examples |
|--------|------------|------------|----------|
| **Starter** | Everyone gets it | Tutorial | Newcomer, Adventurer |
| **Common** | Easy to obtain | Basic milestones | Farmer, Miner, Angler |
| **Rare** | Moderate effort | Achievements | Master Trader, Boss Hunter |
| **Epic** | Challenging | Hard achievements | Solo Legend, Millionaire |
| **Legendary** | Very difficult | Rare achievements | Dragon Slayer, Paradise Creator |
| **Mythic** | Extremely rare | Ultimate challenges | The Completionist, Savior of IRIFA |
| **Event** | Limited time | Seasonal events | Spooky, Winter Spirit |
| **Premium** | Purchase only | Robux store | Golden VIP, Supporter |

## Title Display System

### How to Equip Titles

**Title Menu Access:**
1. Open Main Menu (ESC key)
2. Select "Titles" tab
3. Browse owned titles
4. Click title to equip
5. Title appears above name instantly

**UI Features:**
- **Filter by Category:** Collection, Combat, Economy, Island, etc.
- **Filter by Rarity:** Show only Legendary/Mythic titles
- **Sort Options:** Alphabetical, Date Obtained, Rarity
- **Search Bar:** Quick find by name
- **Preview Mode:** See title above your name before equipping
- **Recent Unlocks:** Last 5 titles obtained highlighted

### Title Display Format

**In-Game Display:**
```
[Title Text]
PlayerName123
```

**Display Properties:**
- Title shown in brackets above name
- Color-coded by rarity:
  - **Common:** White
  - **Rare:** Blue
  - **Epic:** Purple
  - **Legendary:** Gold
  - **Mythic:** Red with glow
  - **Event:** Rainbow gradient
  - **Premium:** Gold with sparkles

**Visibility:**
- Titles visible to all players
- Shows in chat, leaderboards, player list
- Appears when hovering over player

## Developer Notes

### Adding New Titles

**Process:**
1. Define Title ID (TITLE_###)
2. Set display text (keep short, 2-4 words)
3. Assign rarity tier
4. Determine obtainment method
5. Link to achievement/quest/purchase
6. Assign category
7. Add to title database
8. Implement unlock trigger
9. Test display in-game

**Design Guidelines:**
- Keep titles SHORT (2-4 words max)
- Titles should be PRESTIGIOUS
- Avoid generic names ("Player", "User")
- Make titles aspirational
- Balance difficulty with prestige
- Premium titles NEVER give stat bonuses

### Title Color Scheme

**Rarity Colors:**

TitleColors = {
