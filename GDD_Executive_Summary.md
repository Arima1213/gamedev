## 1. Game Overview

### High-Level Concept
**IRIFA** is a multiplayer sandbox life simulation RPG set in a vibrant world of floating islands. Players gather resources, craft powerful equipment, tame magical creatures, build their dream island home, and battle through challenging dungeons while uncovering the mystery of the ancient Heart Gems.

### Genre & Platform
- **Primary Genre:** Multiplayer Sandbox, Life Simulation
- **Secondary Genre:** RPG, Crafting, Creature Collection
- **Platform:** Roblox
- **Target Age:** 10-18 years (ESRB: E10+)
- **Player Count:** Massively multiplayer (persistent world + instanced dungeons)

### Unique Selling Points
1. **Deep Multi-System Integration:** Mining, fishing, farming, combat, taming, crafting all interconnect seamlessly
2. **Creature Companionship:** 30 unique tameable familiars that fight alongside players and provide buffs
3. **Private Island Building:** Fully customizable personal space with 120+ decorations and aesthetic scoring
4. **Rich Story Campaign:** 10-chapter narrative about restoring ancient civilization and defeating the Void
5. **Seasonal Events:** Year-round live service content with exclusive rewards and limited-time content
6. **Fair Monetization:** All gameplay content accessible to free players; premium = convenience and cosmetics

### Design Pillars
1. **"Harvest the World, Forge Your Legend"** - Gathering feeds into meaningful progression
2. **"Your Island, Your Story"** - Personal expression through island building and customization
3. **"Never Alone"** - Companion creatures, NPC relationships, multiplayer dungeons
4. **"Always Something New"** - Seasonal events, daily quests, hidden secrets, live service updates
5. **"Fair for Everyone"** - No pay-to-win, skill and time investment rewarded equally

## 7. Technical Requirements

### Platform Specifications
**Platform:** Roblox  
**Engine:** Roblox Studio (Lua scripting)  
**Target Devices:** PC, Mobile, Console (Xbox)  
**Minimum Requirements:** Roblox standard (Windows 7+, Android 5.0+, iOS 10+)

### Performance Targets
| Metric | Target | Critical |
|--------|--------|----------|
| **Server FPS** | 55-60 | >30 |
| **Client FPS** | 60 | >45 |
| **Max Parts per Area** | <10,000 | <15,000 |
| **DataStore Calls** | <60/minute | <100/minute |
| **Network Send Rate** | <50 KB/s | <100 KB/s |
| **Memory Usage** | <500 MB mobile | <800 MB |
| **Load Time (Join)** | <10 seconds | <20 seconds |

### Architecture Overview
```
Client (Player Device)
    ‚Üì RemoteEvent/RemoteFunction
Server (Roblox Game Server)
    ‚Üì DataStore API
DataStore (Cloud Persistence)
```

**Key Modules:**
- **PlayerDataManager** - Save/load player data
- **InventorySystem** - Item management
- **CraftingEngine** - Recipe processing
- **CombatController** - Damage calculation
- **DungeonInstancer** - Create private instances
- **TradingSystem** - Secure P2P trading
- **WeatherController** - Time/weather sync
- **EventManager** - Seasonal event handling

### Security Measures
1. **Server-Side Validation:** All critical actions (damage, drops, trading) validated server-side
2. **Anti-Exploit:** Sanity checks on all client requests (movement speed, attack rate, resource gain)
3. **Rate Limiting:** Prevent spam (max 10 requests/second per player)
4. **Data Encryption:** Sensitive data hashed/encrypted in DataStore
5. **Admin Tools:** Moderation dashboard for banning, item restoration
6. **Logging:** Track suspicious activity (duplication, speed hacks, item generation)

### Scalability
- **Dungeon Instancing:** Support up to 50 concurrent dungeon instances per server
- **Player Capacity:** 30 players per server (town + open world)
- **Server Sharding:** Multiple game servers for global playerbase
- **Database Optimization:** Batch DataStore calls, cache frequently accessed data
- **LOD System:** Level of detail for decorations, reduce rendering cost

## Glossary of Key Terms

**IRIFA (IRF)** - In-game soft currency, earned through gameplay  
**Robux** - Roblox premium currency, purchased with real money  
**Proficiency** - Skill level in specific activity (Mining, Fishing, Combat, etc.), 1-100  
**Familiar** - Tameable creature companion (30 species, Common‚ÜíMythic)  
**Geode** - Sealed rock containing random minerals (Common‚ÜíMythic)  
**Enhancement** - Permanent upgrade to equipment (+1 to +10, requires materials)  
**Private Island** - Player's personal home space (customizable, 50√ó50m‚Üí200√ó200m)  
**Aesthetic Score** - Island decoration rating (0‚Üí15,000+, unlocks rewards)  
**Heart Gem** - Ancient artifact powering IRIFA (7 gems, main story quest)  
**The Void** - Antagonist force (entropy/corruption, purple-black energy)  
**Valdis/Void Herald** - Main antagonist (7th guardian betrayed for immortality)  
**NPC** - Non-Player Character (quest givers, vendors, story characters)  
**Instance** - Private dungeon copy for individual/group (50 max per server)  
**Trading Board** - Global marketplace for player-to-player item listings  
**Bond Level** - Relationship strength with familiar (0-100, affects stats)  
**Friendship Tier** - Relationship with NPC (Stranger‚ÜíSoulmate, 6 tiers)  
**Paragon Level** - Post-level-100 progression (unlimited, 1M XP each)  
**IRF Sink** - Game mechanic that removes currency from economy (prevents inflation)  
**LOD (Level of Detail)** - Performance optimization (reduce detail at distance)  
**DataStore** - Roblox's cloud save system (player data persistence)
