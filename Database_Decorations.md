## Aesthetic Score System

### Score Calculation

```lua
function AestheticService:CalculateIslandScore(island)
    local totalScore = 0
    local bonuses = {}
    
    -- Base decoration scores
    for _, decoration in pairs(island.Decorations) do
        totalScore = totalScore + decoration.AestheticScore
    end
    
    -- Theme bonuses (if decorations match theme)
    local themeBonus = CalculateThemeBonus(island.Decorations)
    totalScore = totalScore + themeBonus
    
    -- Placement bonuses (symmetry, organization)
    local placementBonus = CalculatePlacementBonus(island.Decorations)
    totalScore = totalScore + placementBonus
    
    -- Rarity bonuses
    local rarityBonus = CalculateRarityBonus(island.Decorations)
    totalScore = totalScore + rarityBonus
    
    -- Set bonuses (matching furniture sets)
    local setBonus = CalculateSetBonus(island.Decorations)
    totalScore = totalScore + setBonus
    
    -- Visitor ratings
    local visitorBonus = island.AverageVisitorRating * 100
    totalScore = totalScore + visitorBonus
    
    return totalScore
end
```

### Theme Bonuses

**Matching Themes:**
- **Japanese Zen:** Cherry blossom tree + Torii gate + Rock garden + Bamboo = +200 bonus
- **Medieval Castle:** Stone walls + Watchtower + Gate + Dragon statue = +250 bonus
- **Tropical Paradise:** Palm trees + Beach setup + Koi pond + Flowers = +150 bonus
- **Fairy Garden:** Mushrooms + Flowers + Gnomes + Magical lights = +180 bonus
- **Modern Luxury:** Luxury furniture + Crystal lights + Fountains = +200 bonus

## Usage Guidelines

### For Players

**Getting Started:**
1. Buy cheap furniture (chairs, tables)
2. Add basic lighting (torches, lamps)
3. Plant trees and flowers
4. Create pathways
5. Build towards a theme

**Maximizing Score:**
- Follow a cohesive theme
- Place decorations symmetrically
- Mix rarity tiers
- Complete matching sets
- Keep island clean and organized

### For Developers

**Adding New Decorations:**
1. Create 3D model
2. Assign ItemID and stats
3. Set aesthetic score (balanced)
4. Choose acquisition method
5. Implement special effects
6. Test placement system
7. Balance cost vs score
